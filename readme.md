## Installation

1. Clone the repository
   ```
   git clone https://github.com/m4dd0c/react-socket-webrtc.git
   ```
2. Install Dependencies
   ```
   cd into cloned repo
   npm run dev-i
   ```
3. Start
   ``` npm run dev ```
